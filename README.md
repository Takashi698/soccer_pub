**タイトル**  
Soccer Pub

**コンセプト**  
サッカー好き達の交流プラットフォーム

サッカーの試合ごとにルームを設定し、情報収集がしやすく、横の繋がりを構築できる場所を提供することが目的。

**バージョン**  
ruby2.6.5  rails 5.2.5

**機能一覧**
* ログイン機能
* ユーザー登録機能
* 文章投稿機能
* 文章編集機能
* 文章削除機能
* 画像投稿機能
* 画像削除機能
* 画像一覧表示機能
* 投稿お気に入り機能
* 投稿コメント機能
* 試合検索機能
* ゲストログイン機能
